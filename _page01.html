Staten Island Ferry Project.

The process.

I created an interactive graphic with an embedded repeating animated gif. I also translated the interactive to become a print graphic.

I started in the early spring, meeting with my Director, Jeff Glick and the editors at the Staten Island Advance to formulate some fresh ideas for presenting a story about Staten Island Ferry Funding and how it's failing. I was also still in a Bootcamp course learning about html/css and javascript coding. I was applying my new skills as I worked on this project.

Once rough designs were sketched, I used Photoshop to create the animated illustration and Illustrator to create the components used within the interactive and print graphics. After that, I used Visual Studio Code to complete the html, css and js coding and Github to house the project files and share renderings with the team. Once that was complete I quickly learned what I could about ArcPublishing to make the files work on our websites.
